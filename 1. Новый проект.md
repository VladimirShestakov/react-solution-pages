# Новый проект

React-Solution работает на Node.js 18 или новее. Проверьте версию командой `node -v` . Если версия старая или Node.js отсутствует, то установите свежую LTS версию. https://nodejs.org/en/download/package-manager 

Создание нового проекта:

```
npm create react-solution@latest
```

Создаётся начальная структура проекта на `React` с использованием фреймворка `React-Solution` и настроенными инструментами `Vite`, `TypeScript`, `Jest`, `eslint`, `prettier`. Файлы с настройками всех инструментов будут в корне проекта.

Если под проект создана новая директория вместо использования текущей, то перейдите в неё (в примере директория project-name).

```
cd project-name
```

В директории проекта выполните команду для загрузки npm пакетов.

```
npm install
```

При необходимости контроля изменений инициализируйте git репозиторий. 

```
git init
```

В терминале откройте свою рабочую директорию (возможно где расположены все ваши проекты). Директорию под новый проект самостоятельно создавать не надо. Всё необходимое созданием одной командой:

```
npm create vite@latest
```

Введите название проекта, оно же будет использовано для названия директории, поэтому желательно использовать символы нижнего регистра без пробелов. Например `example-solution`
Далее выберите фреймворк `React`. И следующим шагом `TypeScript`. 

Будет создана директория под проект (в примере example-solution) и в ней все необходимые файлы и настройки. Перейдите в директорию проекта командой  `cd example-solution` 

Введите команду `npm install` чтобы загрузились все необходимые пакеты. Появится директория `node_modules` Изучать её нам не надо. Там находится код используемых библиотек, в том числе react, vite, typescript.
